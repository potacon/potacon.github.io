한자길 데이터 삭제 안내

Data deletion instructions for 한자길 - 일본어 한자 학습

이 페이지는 한자길 - 일본어 한자 학습 앱의 Google 백업 데이터 삭제 방법을 안내합니다.

Google 로그인은 선택 기능이며, 학습 기록 백업과 복원에만 사용됩니다. 기본 학습 기능은 Google 로그인 없이도 사용할 수 있습니다.

개인정보처리방침: https://potacon.github.io/hanjagil/privacy-policy.html

한국어

앱 내 Google 백업 데이터 삭제 방법

  1. 한자길 앱을 엽니다.
  2. 설정 탭으로 이동합니다.
  3. Google 백업 관리 영역을 확인합니다.
  4. Google 백업 데이터 삭제를 누릅니다.
  5. 삭제 안내를 확인하고 삭제를 확정합니다.

삭제되는 데이터

삭제 기능은 한자길을 위해 Google 로그인 계정에 연결되어 저장된 Cloud Firestore 백업 문서를 삭제합니다.

  • 학습 진행도
  • 오답노트 기록
  • 일일 학습 기록
  • 구간/섹션 퀴즈 완료 기록
  • 현재 학습 section/subsection
  • 마지막 학습 위치
  • 백업에 포함된 앱 설정값이 있는 경우 해당 설정값

삭제되지 않는 데이터

  • 클라우드 백업 삭제는 사용자의 기기에 저장된 로컬 학습 기록을 삭제하지 않습니다.
  • 로컬 데이터는 앱 내 로컬 초기화 기능, 앱 삭제, 또는 Android 설정의 앱 데이터 삭제를 통해 삭제할 수 있습니다.
  • 사용자의 Google 계정 자체는 Google이 관리하며, 한자길 앱에서 Google 계정을 삭제하지 않습니다.

보관 기간 및 재생성

  • 사용자가 앱에서 삭제를 확정하면 Cloud backup data가 삭제됩니다.
  • 삭제 후 한자길은 해당 cloud backup data의 별도 사본을 의도적으로 보관하지 않습니다.
  • 사용자가 나중에 다시 Google 로그인 후 백업 기능을 사용하면 새 백업 데이터가 생성될 수 있습니다.
  • 기기 내부의 로컬 데이터는 개발자가 접근할 수 없습니다.

English

How to delete Google backup data in the app

This page explains how users can delete cloud backup data for Hanjagil - Japanese Kanji Learning.

Google sign-in is optional and is used only for learning record backup and restore. The main learning features can be used without signing in.

  1. Open the app.
  2. Go to Settings.
  3. Open Google backup management.
  4. Tap Delete Google backup data.
  5. Confirm deletion.

Data that will be deleted

The app deletes the Cloud Firestore backup document stored for Hanjagil and associated with the signed-in Google account.

  • Learning progress
  • Wrong-note records
  • Daily study records
  • Subsection quiz completion records
  • Current section/subsection
  • Last study position
  • App settings included in the backup, if present

Data that will not be deleted

  • Local learning records stored on the user's device are not deleted by cloud backup deletion.
  • Users can delete local data from in-app local reset options, by uninstalling the app, or by clearing app data from Android settings.
  • The user's Google Account itself is managed by Google and is not deleted by this app.

Retention and new backups

  • Cloud backup data is deleted when the user confirms deletion in the app.
  • After deletion, the app does not intentionally retain a separate copy of that cloud backup data.
  • If the user signs in again and continues using backup features, new backup data may be created.
  • Local device data is not accessible to the developer.

Additional deletion request

For additional deletion requests, email hun8529@naver.com. Please include the app name and the Google account email used for sign-in.

Other access information

The app does not provide paid or premium content. The app does not use QR codes, PINs, biometric authentication, or membership-only access.
